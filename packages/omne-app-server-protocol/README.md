# @omne/omne-app-server-protocol

TypeScript protocol package generated from Rust single source of truth (`crates/app-server-protocol`).

## What it contains

- `generated/*.d.ts`: JSON-RPC request/notification and event-related protocol types.
- `schema/*.schema.json`: JSON Schema artifacts generated by `omne-app-server`.

## Regenerate

From repo root:

```bash
node ./packages/omne-app-server-protocol/scripts/sync.mjs
```

The script calls:

- `omne-app-server generate-ts --out ...`
- `omne-app-server generate-json-schema --out ...`

If `OMNE_APP_SERVER_BIN` is set, the script uses it directly. Otherwise it falls back to `cargo run -q -p omne-app-server -- ...`.
